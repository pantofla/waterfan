Featurify: Featured Content Gallery Pligg Module

Created By:       CMSTheme Labs
Created On:       01/25/2009
Last Updated:    03/31/2009


Description

A "Featured Content Gallery" Pligg mod that allows you to create an automated rotating image gallery of your stories.


Installation

Open any of your Pligg template files where you want the slidehosw to appear and put this code: 
{$featurify}

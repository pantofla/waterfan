RSS Importer Module

Orignally created by: AshDigg, Modified by Mark Edwards of Simpler Computing and Eric "Yankidank" Heikkinen
Originally Created on: 2007/08/20

Last updated: 2008/09/17

Description

Import RSS feeds from your favorite websites into your Pligg site.

Instructions

  1. Click on "Add a new feed."
  2. Enter feed URL in "Feed URL." input
  3. Configure the feed settings to your preference
  4. Click "Add a new field link" to map the fields of the RSS feed to your Pligg site fields. For example:
    • - link_title
    • - link_url
    • - link_content
